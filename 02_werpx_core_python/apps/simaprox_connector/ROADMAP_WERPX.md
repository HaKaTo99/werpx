- [x] Permission, role, menu, tema

## 6. Ekstensi Fitur
- [x] Custom module untuk kebutuhan spesifik (advanced HR, project portfolio, compliance, dsb)

---

## 📌 Petunjuk Eksekusi
- Ikuti checklist di atas secara bertahap.
- Setiap tahap bisa didokumentasikan di file README/INSTRUKSI masing-masing modul.
- Untuk workflow approval dan custom report, gunakan template yang disediakan di folder ini.
- Untuk integrasi API, gunakan contoh di erpnext_connector.py dan sesuaikan endpoint sesuai kebutuhan.
